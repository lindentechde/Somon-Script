## Static Methods

| Tajik                   | JavaScript        |
| ----------------------- | ----------------- |
| `рӯйхат.аз()`           | `Array.from()`    |
| `рӯйхат.рӯйхатАст()`    | `Array.isArray()` |
| `рӯйхат.азАргументҳо()` | `Array.of()`      |

## Examples

```som
тағ р = [1, 2, 3];
р.илова(4);                    // push(4)
тағ дугона = р.харита(х => х * 2);  // map
тағ филтр = р.филтр(х => х > 1);    // filter
```
